# @ai-masters-community/qwen-agents-code

AI-powered CLI tool optimized for Qwen3-Coder models. A TypeScript-based terminal interface for interacting with Qwen AI models.

## Installation

```bash
npm install -g @ai-masters-community/qwen-agents-code-core
npm install -g @ai-masters-community/qwen-agents-code
```

## Usage

```bash
qwen-agents
```

## Features

- Interactive terminal UI built with React/Ink
- File operations (read, write, edit, search)
- Shell command execution with approval workflow
- MCP (Model Context Protocol) support
- Git integration
- Web search and fetch capabilities

## Requirements

- Node.js >= 20

## Configuration

Settings are stored in `~/.qwen/settings.json`. Use `.qwen-ignore` for file exclusion patterns.
